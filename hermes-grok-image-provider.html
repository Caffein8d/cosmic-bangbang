Hermes Grok Image Provider

Enabling native Grok/xAI image generation support in Hermes Agent through the image_generate tool.

Overview

This project explored adding the ability for agents (like Nova) to choose between different image generation providers (FAL vs xAI/Grok) when calling the image_generate tool.

Key Discovery

Full support for xAI image generation already existed in Hermes under the provider name xai.

  • Provider registered as xai (not grok)
  • Models: grok-imagine-image and grok-imagine-image-quality
  • Uses official xAI endpoint: https://api.x.ai/v1/images/generations

Usage

image_generate(
    prompt="A dramatic supernova...",
    aspect_ratio="landscape",
    provider="xai"        # or "fal"
)

This information is now also documented in the hermes-agent skill.
