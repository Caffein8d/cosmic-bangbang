MCP Integration

Bringing Model Context Protocol servers into the Hermes agent stack for standardized, extensible tool access across all personas.

Overview

MCP (Model Context Protocol) is an open standard for connecting AI agents to external tools and data sources. This project adds first-class MCP support to our Hermes setup so we can leverage standardized servers instead of building everything from scratch.

Why MCP?

Extensibility
Plug in powerful tools (databases, GitHub, Linear, file systems, etc.) without writing custom Hermes tools.
Multi-Persona Ready
Share MCP servers across Artemis, Atlas, Nova, and future personas for consistent capabilities.
Future-Proof
MCP is becoming a standard across the AI agent ecosystem.
Growth Project Alignment
Directly supports the "Tool Power" growth vector by rapidly expanding available capabilities.

Recommended MCP Servers

Initial research on high-value MCP servers for the Growth Project. Focus is on servers that meaningfully extend our current Hermes capabilities (terminal + file tools already strong).

Filesystem
Difficulty: Low
@modelcontextprotocol/server-filesystem (official reference)
Rationale: Secure, controlled access to local files and directories. Highest immediate value for development workflows.
Integration Notes: Very easy via stdio. Can be scoped to specific directories for safety.
Git
Difficulty: Low
mcp-server-git (Python)
Rationale: Read, search, and inspect Git repositories without leaving the agent context. Complements existing terminal git usage.
Integration Notes: Simple to install via uvx or pip. Low risk.
Memory (Knowledge Graph)
Difficulty: Medium
@modelcontextprotocol/server-memory
Rationale: Adds a structured knowledge graph memory layer. Could work alongside our holographic memory system for entity relationships.
Integration Notes: Easy to run, but requires thought on how it interacts with existing memory providers.
GitHub
Difficulty: Medium
Community GitHub MCP servers
Rationale: Full GitHub API access (issues, PRs, repos, actions). High value for development and project management.
Integration Notes: Requires GitHub token. Medium complexity due to auth and rate limits.
Composio or Zapier MCP
Difficulty: Medium-High
Composio-MCP or Zapier MCP
Rationale: Umbrella access to hundreds of SaaS tools (Linear, Slack, Gmail, Notion, Calendar, etc.) through one connection. Massive capability expansion.
Integration Notes: Best long-term option but requires account setup and evaluating auth/security model.
